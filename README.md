# Event Manager App

![App Logo](images/app_icon.png) <!-- ضع مسار أيقونة التطبيق هنا -->

**Event Manager** هو تطبيق Flutter لإدارة الأحداث الشخصية والمناسبات، مع ميزات متقدمة مثل:  

- إنشاء وعرض الأحداث الخاصة بك  
- إضافة صورة بروفايل لكل مستخدم  
- واجهة دردشة بسيطة على التطبيق  
- إعدادات مخصصة للتحكم بالثيم (Dark/Light Mode، Primary Color، Font Size)  
- واجهة جميلة وسهلة الاستخدام  

---

## Features

- **Home Screen:** عرض عدد الأحداث، ترحيب بالمستخدم، إمكانية إضافة صورة بروفايل  
- **Events:** إنشاء الأحداث وعرضها في قائمة  
- **Chat Screen:** دردشة داخلية محدودة على التطبيق  
- **Settings Screen:**  
  - Dark/Light Mode  
  - اختيار اللون الرئيسي للتطبيق  
  - التحكم بحجم الخط  
  - تفعيل/إيقاف الإشعارات  
- **Profile Image Picker:** إضافة صورة من المعرض باستخدام `image_picker`  
- **Bottom Navigation:** للتنقل بين الشاشات بسهولة
